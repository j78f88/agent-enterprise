# Sprints

Template directory for per-sprint planning documents (`SPRINT_NN_PLAN.md`, retros, etc.).

This directory ships empty in agent-homebase. When you consume the library in a project, seed your sprint history file from the starter:

```bash
cp ../../starters/SPRINTS.md ./SPRINTS.md
```

Per-sprint plans created by `@planner` and `@sprint-lead` land here as work progresses.

See [ONBOARDING.md](../ONBOARDING.md) for the full setup flow and [SKILL_FLOW.md](../SKILL_FLOW.md) for how sprint docs are produced and archived.
