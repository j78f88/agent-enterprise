# starters/

Bootstrap templates for adopter projects. Copy them into your project
and fill in the instructional comments with your project-specific
details.

## Files

| File | Purpose |
| --- | --- |
| `BACKLOG_LEDGER.md` | Sprint backlog tracking |
| `BUG_BACKLOG.md` | Bug intake and triage |
| `copilot-instructions.md` | Platform entry point for GitHub Copilot |
| `FILE_HASHES.md` | Integrity verification chain |
| `HANDOFF_REJECTIONS.md` | Loopback protocol log |
| `memory-architecture.md` | Project architecture for `.claude/memory/` |
| `memory-conventions.md` | Project conventions for `.claude/memory/` |
| `NON_GOALS.md` | Explicit scope exclusions |
| `SECURITY_CHANGELOG.md` | Security-relevant change log |
| `SPRINTS.md` | Sprint index and archive links |
